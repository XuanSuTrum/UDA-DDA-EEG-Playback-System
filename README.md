# UDA-DDA EEG Playback System

基于 UDA-DDA 跨被试脑电情绪识别模型的 SEED 离线回放与上位机状态显示系统。

## 项目目标

本项目用于验证 UDA-DDA 模型输出与上位机数据回放、时间同步、状态显示和日志记录模块之间的数据链路。当前验证以 SEED 数据集为例，重点考察窗口级预测结果能否按照连续数据流的时间顺序接入上位机。

本项目不重复评价模型总体分类性能，也不将离线回放结果等同于实时脑电采集或自主刺激控制验证。

## 当前流程

1. 在离线阶段完成 UDA-DDA 模型训练与无监督域适应。
2. 使用目标被试前 3 个试次计算个体化归一化参数。
3. 对后续 12 个试次生成带时间戳的窗口级预测结果。
4. 上位机按照回放时间同步读取脑电数据与预测记录。
5. 界面更新脑电波形、频谱分析、消极/非消极概率、当前状态和系统日志。
6. 独立绘图脚本解析日志并生成连续概率曲线和试次统计。

当前上位机回放模式读取预先生成的预测结果，不在界面线程中实时训练或更新模型。

## 输出说明

模型原始输出包含消极、中性和积极三类概率。面向消极情绪状态观测时，上位机将中性与积极概率合并为非消极概率：

```text
P(non-negative) = P(neutral) + P(positive) = 1 - P(negative)
```

连续结果图中的 50% 虚线仅表示消极与非消极概率相等时的参考位置，不表示刺激触发阈值。

## 数据与隐私

仓库不包含以下内容：

- SEED 原始数据集或其他受许可限制的脑电数据
- 被试级原始数据
- 本地运行日志和完整预测结果
- 大型模型权重与训练检查点
- 包含个人目录信息的 Windows 绝对路径

使用者需要按照相应数据集的官方许可要求自行获取数据。
